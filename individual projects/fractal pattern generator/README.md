
# Fractal Pattern Generator
***
    This project is meant to create a Sierpinski Triangle through the module Turtle in python.

## How to use
***
1. The Pillow library is used for this project, see Installation Instructions for how to install that.
2. Go to main.py in fractal pattern generator in individual projects
3. Run that file!

## Installation Instructions 
***
Before running this project, please make sure to create a new terminal, and type in: pip install Pillow and hit enter.
